# minima-minimask-web-portal
A Minima web portal that connects with MiniMask extension for wallet access payments explorer tools and live network interacti
Simple website to connect MiniMask and interact with Minima.

## Features
- Connect wallet
- Get address
- Get public key
- View balance
- Send payment

## Important
Works only on desktop browser with MiniMask installed.

Download:
https://minimask.org/minimask-2.5.zip
